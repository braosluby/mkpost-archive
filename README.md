# mkpost-archive

## 项目简介

`mkpost-archive` 是一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定域名或网站，仅用于存放和展示零散页面内容，便于长期保存与访问。

## 目录说明

```
mkpost-archive/
├── pages/          # 存放所有独立 HTML 页面文件
├── assets/         # 页面引用的静态资源（样式、图片、脚本等）
├── index.html      # 仓库首页 / 导航页（可选）
└── README.md       # 本说明文件
```

- `pages/` 目录下每个 HTML 文件为一个独立归档单元，文件名通常以日期或简短标识命名。
- `assets/` 用于存放页面所需的公共或专用资源，按需组织子目录。

## 页面归档说明

- 所有页面均为独立 HTML 文档，可直接在浏览器中打开。
- 页面内容可能包含文字、图片、链接等，但均不指向任何具体商业或推广目标。
- 归档页面可能随时间更新或移除，不保证长期可用性。

## 维护说明

- 本仓库由作者手动维护，不定期添加或更新页面内容。
- 欢迎通过 Issue 提出建议或报告链接失效，但无法保证及时响应。
- 不接受外部直接提交的 Pull Request，除非内容与归档主题高度相关。

## 许可

本仓库内容仅供归档与展示，未经允许请勿用于商业用途。  
具体页面可能包含其自身的版权声明，请以实际页面标注为准。
